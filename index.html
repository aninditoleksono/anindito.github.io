Package Siphon untuk mengambil THREDDS data

Siphon adalah suatu modul/package python yang menyediakan banyak perangkat/fungsi untuk mengakes dan mendownload data dari remote data services. Sebagian besar fungsi-fungsi yang tersedia pada siphon dibuat untuk memudahkan akses data yang disimpan dalam THREDDS Data Server (TDS). Selanjutnya kita akan menggabungkan data tersebut untuk mendapatkan prakiraan cuaca 3 hari kedepan dengan menggunakan teknik Time-lag Ensemble.
